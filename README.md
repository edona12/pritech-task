# Pritech Task Manager

A simple React Native mobile application for managing personal tasks.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed or not completed
- View task details
- Search tasks by title
- Filter tasks (All, Todo, Done)
- Store tasks locally using AsyncStorage
- Fetch and display data from a public API
- Simple and clean user interface

## Technologies Used

- React Native
- Expo
- JavaScript
- AsyncStorage
- React Navigation

## Installation

Clone the repository:

```bash
git clone https://github.com/edona12/pritech-task.git
```

Navigate to the project folder:

```bash
cd pritech-task
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npx expo start
```

```text

```

## What Was Implemented

This application allows users to create and manage personal tasks. Users can add new tasks, update their status, delete tasks, search and filter tasks, and view task details. Tasks are stored locally using AsyncStorage to persist data between app sessions. The application also fetches and displays data from a public API.
